Dr. CS codes ...

This site houses the materials for computer science ("CS") courses created and taught by me, Christopher Simpkins ("Dr. CS"), reorganized and enhanced for web or in-person delivery. Most of these courses originated as Georgia Tech courses that I taught to thousands of students, some of these originated as customized workshops for programs in data analytics, quantitative computational finance, and introductory computing. Some of these courses I created out of personal interest or to help crystallize lessons I've learned in professional practice. I am currently a senior software engineer at IBM/The Weather Company building globe-scale cloud-native web services and I have a keen interest in many aspects of computer science and software development.

Computing

Computing for Professionals

Start here, even if you already feel comfortable with computers. The way a programmer or data scientist uses a computer is often quite different from the way ordinary users do. Topics include basic computer organization, operating system usage (especially command line shells), text editors, character encoding, distributed computing (from a user perspective), version control systems, document preparation systems, static web site creation.

Computer Science Fundamentals

Programming Principles

An introduction to programming principles and computation. Suitable for someone who knows the basics of computing and, if mastered, would prepare you for a data structures and algorithms course (typically CS2) at a university. Together, Introduction to Computing and this course cover the material found in a typical university CS 1 course.

Discrete Math Essentials

A short course in the essential mathematics of computer science: logic, proofs, sets, functions, combinatorics and number theory. This course provides a minimal preparation for a university-level course in algorithms and data structures, covering about two thirds of the content of a typical university discrete math course.

Intensive Python (WIP)

A short course in Python for people who already know how to program, perhaps in another language. Covers all the basic features of Python in greater depth than would be taught in a first course in programming. This course prepares students well for work that requires Python, such as data analytics using the Python stack or software engineering using Python. Well-suited for workshops or "boot camps."

Data Manipulation

The acquisition, representation, transformation, storage and presentation of data with an introduction to the tools of data analytics. Topics include file I/O, text data storage and exchange formats, text processing, web mining, SQL databases, and essential elements of the SciPy stack -- NumPy, Pandas, and Matplotlib. Requires Python skill at the level of Intensive Python.

An introduction to machine learning covering the material typically taught in a university senior undergraduate or graduate introduction to machine learning.
